# Glossary

| Term | Meaning in this lab |
|---|---|
| AND type | すべてのfieldを持つrecord/tuple |
| OR type | いずれか一つのcaseを持つdiscriminated union |
| Smart constructor | 制約を確認してdomain typeを返すfunction |
| Invariant | 常に守るべきbusiness rule |
| Deriver | input dataからchange/outcomeを導くpure function |
| Controller | data取得とeffectを調整するimperative shell |
| Port | Applicationが必要とする外部能力のtype |
| Adapter | Portの具体実装 |
| Railway | `Result`を使った成功/失敗pipeline |
| Bounded Context | 一つのmodelと言語が一貫して有効な境界 |
| Illegal state | Business上存在してはいけないdata combination |
| Composition Root | 実装とdependencyを組み立てる場所 |
