# 用語集

| 用語 | このラボでの意味 |
|---|---|
| ANDタイプ | すべてのフィールドを持つレコード/タプル |
| ORタイプ | いずれか一つのケースを持つ識別されたユニオン |
| スマートコンストラクタ | 制約を確認してドメインタイプを返す関数 |
| 不変条件 | 常に守るべきビジネスルール |
| デライバー | 入力データから変更/結果を導く純粋関数 |
| コントローラー | データ取得と効果を調整する命令型シェル |
| ポート | アプリケーションが必要とする外部能力のタイプ |
| アダプター | ポートの具体的な実装 |
| レールウェイ | `Result`を使った成功/失敗のパイプライン |
| バウンデッドコンテキスト | 一つのモデルと言語が一貫して有効な境界 |
| 不正な状態 | ビジネス上存在してはいけないデータの組み合わせ |
| コンポジションルート | 実装と依存関係を組み立てる場所 |
